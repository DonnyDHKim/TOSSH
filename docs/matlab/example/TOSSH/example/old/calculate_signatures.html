calculate_signatures

PURPOSE

calculate_signatures Interactive signature calculation. We use cell arrays

SYNOPSIS

function [results] = calculate_signatures(Q_mat, t_mat, P_mat, varargin)

DESCRIPTION 

calculate_signatures Interactive signature calculation. We use cell arrays
 to allow for time series of different lengths.

    INPUT
   Q_mat: streamflow Q [mm/timestep] time series as a cell matrix of the
   following format: {Q_1; Q_2; ...; Q_n}
   t_mat: date t [Matlab datenum] time series as a cell matrix of the
   following format: {t_1; t_2; ...; t_n}
   P_mat: precipitation P [mm/timestep] time series as a cell matrix of the
   following format: {P_1; P_2; ...; P_n}
    OPTIONAL
   signature_selection: which set of signatures to calculate. (For more
   information on the selections please have a look at the documentation.)

   OUTPUT
